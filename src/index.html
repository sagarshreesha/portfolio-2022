Projects

Tggr - Tag Based File Sharing

Tired of digging for that file or link you had stored somewhere ? Want to quickly access your files cross platform ? Upload files under a common tag and collaborate like never before.
Suppose a professor wants her students to submit their assignments digitally. All she has to do is create a tag like say '6th-sem-assignments' and share it with her students. The students can upload the files under that tag.
Built using React and Firebase Cloud Functions. Future plan is to build it using a microservice architecture and deploy it using Kubernetes.

PU Results | Govt. PU College

Due to COVID-19 lockdown colleges were asked not to display the results at the college, but to send it to them directly. The best solution to this was to build a Result Portal where they could check their results with their Reg. No.
So I built this portal for over 10,000 students to check their results. Each student had around 15 parameters. The DB was generated in MongoDB and I created a REST API with GraphQL queries. The frontend was built using Vue as it's easier and the development process is quicker.
